Что такое массив?

Массив — это коллекция (объект встроенного языка), содержащая пронумерованную последовательность значений произвольного типа. Каждому элементу присваивается последовательный целочисленный номер — идекс, начинающийся с нуля. Благодаря индексам можно быстро получить доступ к любому элементу или изменить его. Массивы широко используются для временного хранения данных, их группировки и передачи между процедурами. В 1С:Предприятие существует два типа массивов: обычный (изменяемый) и фиксированный (неизменяемый). Обычный массив имеет динамический размер: количество элементов можно менять в процессе выполнения программы. Фиксированный массив, наоборот, создаётся с заданным размером, и его нельзя изменить. В этом уроке мы сосредоточимся на обычных массивах.

Создание массива

Создать массив можно несколькими способами.

Основные методы работы с массивом

У массива есть несколько ключевых методов для управления элементами.

ОписаниеПример
Метод Добавить(Значение) добавляет элемент в конец массива. М.Добавить("строка")
Метод Вставить(Индекс, Значение) вставляет элемент по указанному индексу, сдвигая остальные вправо. М.Вставить(0, 100)
Метод Удалить(Индекс) удаляет элемент по индексу. М.Удалить(2)
Метод Очистить() удаляет все элементы из массива. М.Очистить()

Получение информации о массиве

Для получения сведений о массиве используются следующие методы и свойства.

ОписаниеПример
Метод Количество() возвращает количество элементов в массиве. Кол = М.Количество()
Метод ВГраница() возвращает наибольший индекс массива (Количество - 1). ВерхняяГраница = М.ВГраница()
Метод Найти(Значение) ищет элемент в массиве, возвращает его индекс или Неопределено. Индекс = М.Найти("искомое")

Доступ к элементам: чтение и запись

Получить или изменить значение элемента можно с помощью квадратных скобок с указанием индекса или методов Получить и Установить.

Обход массива (перебор элементов)

Чаще всего элементы массива обрабатываются в циклах.

Многомерные массивы

В 1С можно создавать многомерные массивы — массивы, элементами которых являются другие массивы.

Фиксированный массив (неизменяемый)

В 1С также существует тип «Фиксированный массив». В отличие от обычного, его размер и значения нельзя изменить после создания. Он полезен, когда нужно защитить данные от случайного изменения или передать коллекцию на клиент/сервер.

Массивы структур

Массив может хранить элементы любого типа, включая «Структуру». Это позволяет создать коллекцию, где каждый элемент является набором именованных полей (пар «Ключ-Значение»).

Сравнение массивов

Массивы можно сравнивать между собой. Два массива считаются равными, если их размеры совпадают и элементы на соответствующих индексах равны.

1C Rush > Преобразование типов

Зачем нужно преобразовывать типы?

В 1С:Предприятие используется динамическая типизация — переменная сама получает тип в момент присваивания значения. Однако часто возникает необходимость явно преобразовать данные из одного типа в другой: строку — в число для расчётов, число — в строку для вывода сообщения, дату — в строку для отчёта и т.д. Также платформа выполняет неявные преобразования в арифметических операциях, когда операнды имеют разные типы.

Функции явного преобразования

Встроенные функции преобразования позволяют безопасно привести значение к нужному типу.

1. Преобразование в число

Функция Число() преобразует строку или булево значение в число. Если строка содержит целое или дробное число (разделитель – точка или запятая), преобразование успешно. Истина превращается в 1, Ложь – в 0. Недопустимые строки вызывают ошибку.

2. Преобразование в строку

Строка() преобразует любое значение в его строковое представление согласно региональным настройкам. Для чисел она вставляет пробелы между разрядами, что не всегда удобно. Функция Формат() даёт полный контроль над форматом.

3. Преобразование в дату

Функция Дата() преобразует строку в дату. Строка обязательно должна быть в формате 'ГГГГММДДЧЧММСС' (часть с временем можно опустить). Также существует вариант передачи числовых компонентов: Дата(Год, Месяц, День, Час, Минута, Секунда).

4. Преобразование в булево

Функция Булево() преобразует число, строку или булево значение в тип Булево. Число 0 даёт Ложь, любое другое число – Истина. Строки "Да", "Истина" преобразуются в Истина, строки "Нет", "Ложь" – в Ложь (без учёта регистра).

Неявные преобразования в выражениях

В арифметических операциях (+, -, *, /) и сравнениях 1С может автоматически преобразовывать типы. Тип результата обычно определяется по первому операнду. Ниже приведены основные правила.

Тип первого операнда Добавление операнда Вывод
"абв" (Строка) "абв" + 2 Строка: "абв2"
"2" (Строка) "2" + 2 Строка: "22"
"2" (Строка) "2" - 2 Число: 0
"2" (Строка) "2" / 2 Число: 1
2 (Число) 2 + Истина Число: 3 (Истина=1)
2 (Число) 2 * Ложь Число: 0
2 (Число) 2 + "2" Число: 4
'20250606' (Дата) '20250606' + 86400 Дата: 07.06.2025 00:00:00
'20250606' (Дата) '20250606' - '20250605' Число: 86400
Истина (Булево) Истина + 2 Число: 3
